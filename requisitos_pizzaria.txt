=== Regras de Negócio ===
Identificador	Descrição
RN01	E-mail do usuário deve ser único
RN02	Senha deve ter no mínimo 8 caracteres e conter letras e números
RN03	Login só é permitido com e-mail e senha válidos
RN04	Rotas protegidas exigem token de autenticação válido
RN05	Usuário só pode ver/alterar os próprios dados; admin pode ver todos
RN06	Somente administrador pode cadastrar produtos
RN07	Listagem por categoria exige categoria existente e retorna só produtos dessa categoria
RN08	Somente administrador pode criar categorias; nome não pode se repetir
RN09	Listar todas as categorias retorna categorias ativas, ordenadas por nome
RN10	Um pedido é criado com status "ABERTO" e vinculado ao usuário criador
RN11	Deletar pedido só é permitido se o status for "ABERTO" e o pedido for do próprio usuário
RN12	Adicionar item só é permitido em pedidos "ABERTO"; item deve referenciar produto válido e quantidade > 0
RN13	Remover item só é permitido em pedidos "ABERTO" do próprio usuário
RN14	Enviar pedido exige pelo menos 1 item; ao enviar, o status muda para "ENVIADO"
RN15	Listar pedidos: usuário vê apenas os seus; administrador pode ver todos
RN16	Detalhes do pedido: acesso permitido apenas ao dono do pedido ou administrador
RN17	Finalizar pedido só se o status for "ENVIADO"; após finalizar, não permite alterações e grava data/hora
